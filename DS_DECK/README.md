# ⌨️ DS DECK (DeskStream Deck)

ESP32-C3와 4개의 OLED를 이용한 독립형 블루투스 매크로 패드 프로젝트입니다.

## 🛠 필수 개발 환경
- **Board Manager**: ESP32 by Espressif Systems (v3.x 이상 권장)
- **Board Selection**: ESP32C3 Dev Module

## 📚 필수 라이브러리 목록 (모두 설치 필수)

### 1. 라이브러리 매니저에서 설치 (검색 가능)
| 라이브러리명 | 용도 | 
| :--- | :--- |
| **U8g2** | 4개의 OLED 디스플레이 제어 | 
| **HijelHID\_BLEKeyboard** | 최신 ESP32 3.x 환경용 BLE 키보드 (NimBLE 기반) |
| **ArduinoJson** | 기기 설정값 관리 |

*(Phase 3 개발 단계로 접어들며 WiFi 및 대용량 JSON 처리가 포함되었습니다. 컴파일 시 용량 에러를 방지하기 위해 4096바이트 DynamicJson 구조를 채택했습니다.)*

---

## 🚀 하드웨어 핀 맵 (DeskStream Board)
- **Buttons**: GPIO 1, 4, 10, 9
- **I2C (HW)**: SDA(5), SCL(6) - OLED 1, 2
- **I2C (SW)**: SDA(2), SCL(3) - OLED 3, 4
- **Buzzer**: GPIO 7


---

## 📖 상세 사용 방법 (Usage Guide)

### 1. 초기 Wi-Fi 및 네트워크 설정
기기를 처음 켜거나 네트워크를 변경할 때 다음 단계를 따르세요.
1. 기기 전원을 켜면 OLED에 `Wi-Fi Setup` 관련 메시지가 출력됩니다.
2. 스마트폰이나 PC에서 Wi-Fi 목록을 검색하여 **`DS_DECK_SETUP`**에 접속합니다.
3. 자동으로 열리는 설정 페이지에서 현재 사용 중인 공유기의 SSID와 비밀번호를 입력합니다.
4. 연결이 완료되면 OLED에 기기의 **IP 주소**가 표시됩니다.

### 2. 블루투스 페어링 (PC/Mac 연결)
Wi-Fi 설정과 별개로, 키보드 기능을 사용하기 위해 블루투스 연결이 필요합니다.
1. PC 또는 Mac의 블루투스 설정 창을 엽니다.
2. 장치 찾기에서 **`DS DECK`**이라는 이름의 기기를 선택합니다.
3. 연결됨 상태가 되면 즉시 키보드 매크로를 사용할 수 있습니다.
   - *팁*: 연결이 끊겼을 경우 기기를 재부팅하거나 PC의 블루투스를 껐다 켜면 자동으로 재접속됩니다.

### 3. 웹 대시보드 접속 (환경 설정)
IP 주소가 확인되면 웹 브라우저를 통해 매크로를 자유롭게 설정할 수 있습니다.
- 브라우저 주소창에 OLED에 표시된 IP(예: `192.168.0.22`)를 입력합니다.
- **DS DECK CONTROL PANEL**가 열리며, 4개 버튼의 라벨, 모드, 매크로 내용을 실시간으로 수정할 수 있습니다.
- **Save & Apply** 버튼을 누르면 기기에서 "삐-" 소리와 함께 즉시 설정이 반영됩니다.

### 4. 버튼 동작 모드 및 규칙 (macOS 최적화)
- **일반 텍스트 (Macro)**: 입력한 긴 문자열을 타이핑합니다.
- **브라우저 모드 (Browser)**: 등록된 URL을 바탕으로 macOS의 Spotlight를 소환, 곧바로 사용자의 기본 웹 브라우저를 열어줍니다. (⚠️ 브라우저 런처 사용 시 맥의 상태가 **'영문 입력'** 상태여야 원활하게 Spotlight 주소 인식이 이뤄집니다.)
- **한/영 전환 토큰**: 일반 문자열 중간에 **`[#CAPS#]`**를 기입하면 macOS 표준 단축키(`Ctrl + Space`)가 자동으로 수행됩니다.
  - *예시*: `Hello[#CAPS#]안녕` -> Hello 입력 후 자동으로 한/영 전환 뒤 안녕 입력.
- **단축키 (Combo) 모드**: `CMD`, `Shift`, `Alt` 등 조합키와 `F1~F12`, 방향키 등을 선택하여 복합 단축 명령을 수행할 수 있습니다.

### 4. 사운드 및 가시성 피드백
- **부저 알림**: 설정이 저장되거나 버튼이 눌릴 때 짧은 부저음이 발생하여 작동 상태를 귀로 확인 가능합니다.
- **OLED 상태창**: 각 버튼의 현재 기능(라벨 또는 미디어 아이콘)이 개별 OLED에 뚜렷하게 표시됩니다.

## ⌨️ 한/영 자동 전환 고도화 (macOS 전용)

DS DECK은 macOS 환경에서 언어 상태에 관계없이 완벽한 타이핑을 지원하기 위해 **Karabiner-Elements**와 연동하여 동작합니다.

### 1. Karabiner 프로젝트 설정 (필수)
프로젝트 폴더 내의 [`macro_language.json`] 파일을 아래 경로로 복사합니다.
*   **복사 경로**: `~/.config/karabiner/assets/complex_modifications`

### 2. Karabiner 앱에서 규칙 활성화
1. **Karabiner-Elements** 앱을 실행합니다.
2. 좌측 메뉴에서 **Complex Modifications** 탭을 클릭합니다.
3. 하단의 **Add rule** 버튼을 누릅니다.
4. 목록에서 **"매크로 키보드 한/영 강제 할당"** 항목을 찾습니다.
5. 우측의 **Enable** 버튼을 클릭하여 설정을 활성화합니다.

## ⚡ 주요 특징 (Premium Features)
- **완벽한 한/영 전환 엔진**: 단순 토글이 아닌 `F19`(영어), `F20`(한글) 강제 할당 방식으로 오타 없는 타이핑 보장.
- **편리한 매크로 입력**: 자주 사용하는 문장이나 복합 단축키를 버튼 하나로 입력.
- **macOS 완전 호환**: Karabiner 연동을 통한 최적화된 입력 환경 제공.
- **Dirty-Checking 디스플레이**: 변경된 픽셀만 전송하는 알고리즘으로 OLED 수명 연장 및 반응성 향상.

## ⚠️ 주의 사항
- 본 프로젝트는 **macOS** 사용 환경에 최적화되어 있습니다.
- 윈도우 사용자의 경우 한/영 전환을 위해 별도의 OS 설정이 필요합니다. (향후 업데이트 예정)
- 컴파일 시 `Partition Scheme`을 **`Huge App`** 또는 **`No OTA (2MB APP/2MB SPIFFS)`**로 설정해야 합니다.
