# ESP32-C3 4-OLED 스마트 정보 스테이션 (Smart_Info_Station)

이 프로젝트는 **MakerGO ESP32 C3 SuperMini** 보드를 사용하여 4개의 OLED 디스플레이를 독립적으로 제어하고, **시각, 날씨, 미세먼지, 유튜브 구독자 수, 증시(코스피, 코스닥, S&P 500), 비트코인(BTC), 원/달러 환율** 등 다양한 정보를 실시간 모니터링하는 데스크탑 정보 표시기입니다. 

## 🚀 주요 기능 및 특징

1.  **하이브리드 I2C 아키텍처 (안정성과 속도)**
    - **OLED 1, 2 (Hardware I2C)**: 1초마다 변경되는 시간을 가장 부드럽고 지연 없이 출력하기 위해 고속 하드웨어 컨트롤러를 배정했습니다.
    - **OLED 3, 4 (Software I2C)**: 화면 갱신 빈도가 낮고 안정적인 디스플레이 유지를 위해 독립적인 소프트웨어 I2C로 물리적 충돌을 차단했습니다.
2.  **FreeRTOS 태스크 기반 비동기 데이터 갱신 (Non-blocking)**
    - 서버에서 **날씨**, **먼지**, **유튜브**, **증시** 데이터를 가져오기 위해 수 초가 걸리는 작업을 백그라운드 태스크(`dataTask`)로 완전히 분리했습니다.
    - 이에 따라 데이터를 갱신하는 순간에도 **OLED의 시계나 화면 표시는 결코 멈추지 않고 즉각적으로 반응**합니다.
3.  **데이터 수집 효율화 및 지능형 업데이트 정책 (v2.2.1)**
    - **가시성 기반 파싱**: 현재 OLED 화면에 표시되지 않는 위젯은 불필요하게 서버를 호출하지 않습니다. (단, 부팅 시 최초 1회는 데이터를 미리 가져옴)
    - **금융 지수 시간 최적화**: 
      - **국내 증시(KOSPI, KOSDAQ, KPI200)**: 평일(월~금) 08:00~18:00 사이에만 작동.
      - **미국 증시(S&P 500, NASDAQ)**: 한국 시간 기준 평일(월~토 오전) 17:00~익일 09:00 사이에만 작동.
    - **수집 중 기존 데이터 유지**: 데이터를 가져오는 동안에도 화면이 0으로 초기화되거나 "Updating..."으로 덮이지 않고, 이전 성공 데이터를 안정적으로 유지합니다.
4.  **강력한 웹 대시보드 및 실시간 설정**
    - 공유기 설정, 유튜브 채널, 미세먼지 측정소, 타임존 등을 소스 코드 수정 없이 웹 브라우저에서 실시간으로 변경 가능합니다.
    - **통합 화면 배치 시스템**: 실제 OLED 배치에 맞춘 가로 4열 UI를 통해 슬롯별 위젯을 직관적으로 구성할 수 있습니다.
5.  **안정성 및 영구 저장**
    - **NVS 영구 저장**: 모든 설정은 ESP32 내부에 안전하게 보관되어 전원 재시작 후에도 유지됩니다.
    - **전방위 Retry 시스템**: 모든 데이터 위젯에 최대 3회 재시도 및 상태 자동 초기화 로직을 적용하여 구동 안정성을 극대화했습니다.
6.  **사용자 인터페이스**
    - 버튼 하나로 1페이지(기본 정보)와 2페이지(금융/추가 정보)를 손쉽게 전환할 수 있습니다.

---
> 💡 상세한 업데이트 내역 및 버전별 수정 사항은 [RELEASE_NOTES.md](./RELEASE_NOTES.md)에서 확인하실 수 있습니다.
7.  **Direct Register Access를 통한 SW I2C 극한 가속**
    - 기본 U8g2 라이브러리의 무거운 아두이노 함수 통신 오버헤드를 우회하기 위해 **ESP32-C3 전용 네이티브 HAL 기반 커스텀 콜백 함수**(`u8x8_gpio_and_delay_esp32_c3_fast`)를 자체 구현했습니다.
    - 통신 속도를 **22kHz에서 약 360kHz(Fast Mode 규격 근접)까지 무려 15배 이상 폭발적으로 향상**시켰습니다.

## 🛠 하드웨어 구성 요소

-   **Main Board**: MakerGO ESP32 C3 SuperMini (또는 범용 ESP32-C3 보드)
-   **Display**: 0.96 inch SSD1306 OLED 모듈 (128x64px) x 4개
-   각 모듈은 I2C 주소 `0x3C`와 `0x3D`로 구성된 2개 쌍을 각각 다른 핀(5,6번 / 8,9번) 버스로 연결합니다.

## ⚙️ 아두이노 IDE 소프트웨어 설정 주의사항

1. **Board Manager**: 보드는 `ESP32C3 Dev Module` 등을 사용합니다.
2. **Partition Scheme**: `Huge APP (3MB No OTA / 1MB SPIFFS)` 를 **반드시** 선택해야 합니다. 프로젝트에 포함된 한글 글꼴(`u8g2_font_unifont_t_korean2`)이 기본 제공 파티션 용량(~1.25MB)을 초과하기 때문입니다!

### 필수 라이브러리 목록
- `U8g2` (강력한 그래픽 및 OLED 폰트 라이브러리)
- `WiFiManager` (AP 모드를 통한 공유기 연결)
- `UrlEncode` (HTTP 웹 요청 URL 변환 라이브러리)

## 📁 위젯별 파일 구조 분석

- `Smart_Info_Station.ino` : 웹 서버, 스크린맵, 디스플레이 통신, 9번 핀 스위치 제어, 비동기 데이터 태스크 메인 코어.
- `weather.h` : 기상청 공개 API 데이터 단기 예보 파싱 로직.
- `dust.h` : 네이버 모바일 미세먼지 크롤링 (PM10, PM2.5 파싱 및 안정화 예외 처리 적용).
- `youtube.h` : 유튜브 데스크톱 페이지 크롤링 기반 구독자 수 수집 기능.
- `market.h` : 네이버 금융 API 연동 모듈 (KOSPI, KOSDAQ, S&P 500, NASDAQ, **선물지수, 코스피200** 통합).
- `btc.h` : 바이낸스 API 연동 (비트코인 달러 가격 및 등락률).
- `usdkrw.h` : 야후 파이낸스 API 기반 원/달러(USD/KRW) 실시간 환율 파싱.

## 📝 최초 시작 방법 (Quick Start)

1. 코드를 성공적으로 플래시 후, 전원을 인가하면 1번 화면에 **"WiFi Connecting..."** 메시지가 나타납니다.
2. PC 혹은 스마트폰의 Wi-Fi 설정에서 **`Smart_Info_Station`** 핫스팟에 연결합니다.
3. 인증 화면(혹은 `192.168.4.1`)을 통해 집이나 사무실의 공유기 정보를 입력하세요.
4. 연결 과정이 끝나면, 1번 화면 하단에 새로 할당받은 **IP 주소(예: `192.168.0.x`)**가 표시됩니다.
5. 동일 네트워크 망에서 스마트폰/PC의 웹 브라우저로 해당 IP 주소에 접속하면 각종 로컬 정보, 타깃 유튜브 채널, 거주 국가의 **시간대(Timezone offset)**를 자유롭게 설정할 수 있습니다!
