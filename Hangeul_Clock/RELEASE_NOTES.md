## [v2.4.4] - 2026-04-12 22:05
### 🚀 시스템 최적화 및 안정화 (Architectural Polish & stabilization)
- **제로-스터터(Zero-Stutter) 웹 API**: 폰트 슬롯 이름 **메모리 캐싱 시스템**을 도입하여, 웹 대시보드 호출 시 시스템이 5초간 멈추던 LittleFS I/O 병목을 근본적으로 해결했습니다.
- **애니메이션 잔상 및 멈춤 방지**: `refreshNow()` 보호 로직을 통해 폰트 전환(로딩) 중 문자가 애니메이션 중간 프레임에 멈춰있던 시각적 결함을 완벽히 수정했습니다.
- **웹 미리보기 색상 반전 연동**: '색상 반전' 설정이 웹 대시보드의 실시간 캔버스 미리보기에도 즉각 반영되도록 렌더링 스크립트를 확장했습니다.
- **설정 영속성 완결**: `is_inverted`(반전 모드) 및 `font_slot` 필드를 NVS 저장 대상에 추가하여 재부팅 후에도 모든 사용자 설정이 유지되도록 보강했습니다.
- **아키텍처 정규화**: `DisplayManager` 클래스를 생성자 기반 초기화 구조로 정돈하여 유지보수성과 컴파일 안정성을 확보했습니다.

### 🏛️ 아키텍처 프로덕션 등급 고도화 및 UI/UX 정예화 (Architectural Refinement & UI Fixes)
- **시스템 서비스 유닛화 (Step 5)**:
    - **`InputManager` 도입**: 버튼 입력 및 인터럽트 로직을 클래스로 캡슐화하여 메인 코드의 의존성을 획기적으로 낮췄습니다.
    - **`Logger` 도입**: OLED 1번 화면과 시리얼 모니터에 동시 출력되는 지능형 로거를 구축했습니다. (시리얼 출력 시 점`.` 애니메이션 줄바꿈 방지 최적화 포함)
- **UI/UX 감성 품질 및 버그 수정**:
    - **"정각" 레이아웃 정합성**: "10시 10분 정각" 등 모든 상황에서 "정각"이 누락되지 않도록 로직을 보충하고, OLED 화면 정중앙 배치 엔진을 고도화했습니다.
    - **화면 전환 텍스트 겹침 해결**: BTN4를 눌러 IP 주소나 도움말 확인 시 이전 텍스트(초 단위 등)가 남던 버그를 `clearAll` 연동으로 완벽히 해결했습니다.
    - **OLED 잔상(Ghosting) 원천 차단**: 섀도우 버퍼 동기화(`pushParallel`)를 로거에 연동하고, 부팅 종료 시 이전 세션의 잔상을 완벽히 소거합니다.
    - **시보 아이콘 위치 고정**: "정각" 중앙 배치 시 다른 화면에 종 아이콘이 나타나던 버그를 수정하여 1번 화면(Title)에만 나타나도록 제한했습니다.
- **기능 및 로직 고도화**:
    - **24시간제 수사 표준화**: 13시 이상 시간 표기 시 한자어 수사(십삼 시 등)를 적용하는 로직을 C++ 백엔드에 완성했습니다.
    - **지능형 로깅 엔진**: 시리얼 모니터 점(.) 애니메이션 줄바꿈 방지 알고리즘을 탑재하여 로깅 가독성을 혁신했습니다.
- **Font Studio 리소스 확장**:
    - 웹 대시보드에서 "**정**", "**각**" 캐릭터를 직접 추출 및 업로드 가능하도록 리스트를 확장하고, 브라우저 미리보기 로직을 하드웨어와 1:1 동기화했습니다.
- **코드 무결성**: 메인 루프(`ino`)를 200라인 이내로 최적화하고 펌웨어 버전 표기를 v1.9.0으로 일원화했습니다.

- **자연스러운 글자 겹침(Overlap) 구현**: 32px의 글자 간격은 유지하되, 렌더링 시 ±16px 오프셋을 허용하여 넓은 글자가 옆 칸을 자연스럽게 침범하는 고급 렌더링 효과를 적용했습니다.
- **메모리 캐시 최적화**: 64px 규격에 맞춰 `CachedChar` 구조체를 업데이트하고 LittleFS 파일 읽기 로직을 512바이트 단위로 최적화했습니다.
