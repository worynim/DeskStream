# Release Notes - Korean Clock

## [v1.3.52] - 2026-04-11
### 🚀 최적화 및 안정화 (Stability & Performance)
- **화면 잔상(Ghosting) 완전 해결**: `clearAll()` 메서드를 도입하여 모드 전환 시 이전 픽셀 잔상을 물리적으로 소거.
- **I2C 대역폭 최적화**: 중복 전송 명령을 제거하고 `pushParallel()` 배경 태스크로 전송 로직을 단일화하여 성능 향상.
- **상수 표준화**: 모든 모드(Animation, Display, Format)를 0-based 인덱스로 통일하여 웹-기기 간 데이터 정합성 확보.
- **명칭 명확화**: 코드 내 모든 `ANIM_` 약어를 `ANIMATION_` 풀네임으로 교체하여 가독성 및 유지보수성 증대.

### ✨ UI/UX 개선
- **도움말 상태 표시**: 버튼 도움말 페이지에서 시보(CHIME) 및 플립(FLIP)의 실시간 ON/OFF 상태 확인 가능.
- **실시간 UI 갱신**: 도움말이나 IP 확인 화면 중에도 버튼 조작 시 정보가 즉각적으로 다시 그려지도록 개선.
- **폰트 즉시 반영**: 웹에서 폰트를 업로드하는 즉시 시계의 모든 글자가 새 폰트로 자동 갱신.

### 🌐 Web Dashboard 고도화
- **실시간 동기화 (Polling)**: 기기에서 물리 버튼으로 설정을 바꾸면 3초 내에 웹 화면에 자동 반영 (새로고침 불필요).
- **UI 정돈**: 동작하지 않던 '폰트 초기화' 버튼을 삭제하고 업로드 프로세스를 간소화.

## [v1.3.0] - 2026-04-11
### Advanced Display Modes & 24H Format
- **숫자 모드 (Numeric Mode)**: '열두시 삼십분' 대신 '12시 30분' 형태로 표시하는 모드 추가
- **24시간제 지원**: 12H(오전/오후)와 24H(0~23시) 형식을 선택 가능하도록 구현
- **지능형 날짜 표시**: 24시간제 모드 사용 시 첫 번째 OLED에 자동으로 날짜(예: '13일' 또는 '십삼일') 표시
- **제로 패딩 (Zero Padding)**: 숫자 모드에서 10 미만의 분, 초 표시 시 '05분'과 같이 0을 채워 시인성 개선
- **Web UI 확장**: Font Studio에서 모든 표시 모드와 시간 형식을 실시간으로 제어 및 시뮬레이션 가능

## [v1.2.0] - 2026-04-11
### Intelligence Animation Engine
- **지능형 차분 업데이트**: 변경된 글자만 감지하여 개별적으로 애니메이션 처리하는 엔진 도입
- **Scroll Up 효과**: 아래에서 위로 부드럽게 글자가 올라오는 애니메이션 구현 (프레임당 8px)
- **설정 영속성**: `Preferences`를 사용하여 애니메이션 모드 설정을 비휘발성 메모리에 저장
- **Web Simulation**: Font Studio에서 애니메이션 모드를 선택하고 브라우저에서 실시간으로 시뮬레이션 가능
- **Configurable Speed**: `config.h`에서 애니메이션 속도(`ANIMATION_STEP_DELAY_MS`) 조절 가능

## [v1.1.0] - 2026-04-10
### Character-Based Font Engine & Studio
- **낱자(낱자) 기반 폰트 최적화**: 134개의 단어 이미지 대신 **28개의 고유 낱자** 저장 방식으로 전환하여 메모리 및 업로드 효율 극대화
- **지능형 레이아웃 엔진**:
    - **오전/오후**: 화면 정중앙 배치
    - **시, 분, 초**: "시/분/초" 단위를 우측 끝에 고정하고, 앞의 숫자들을 남은 공간의 중앙에 배치하는 가변 정렬 구현
- **Font Studio v2 (Web)**: 폰트 파일(.ttf, .otf)을 웹 서버(`/`)에서 직접 시계용 낱자 비트맵으로 변환 및 일괄 업로드 기능 제공
- **Fixed Graphics Noise**: U8g2 호환 가로형(Horizontal MSB) 인코딩 적용으로 출력 품질 개선

## [v1.0.0] - 2026-04-10
### Initial Release (MVP)
- **Core**: NTP 시간 동기화 및 대한민국 표준시(KST) 적용
- **Korean Logic**: 숫자를 한글로 변환하는 엔진 구현
- **High-Performance Display**: 1MHz I2C 전송 및 병렬 태스크 가속
