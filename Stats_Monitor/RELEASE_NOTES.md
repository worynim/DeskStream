# 📝 Release Notes - Stats_Monitor

## [v1.4.2] - 2026-04-05
### 🚀 Improved
- **밝기 설정 최적화**: 매직 넘버로 관리되던 밝기 설정값을 배열(`BRIGHTNESS_LEVELS`) 구조로 개선하여 코드 유지보수성을 높였습니다.
- **밝기 수치 세밀화**: 사용자 요청에 맞춰 4단계 밝기 값을 **1, 50, 100, 240**으로 새롭게 조정하였습니다.
- **클라이언트 트레이 UI 및 브랜딩 강화**: 
    - 시스템 트레이 아이콘의 종료 메뉴 이름을 'Quit'에서 '**Desk Stream Stats Monitor Quit**'으로 변경하여 식별성을 높였습니다.
    - 트레이 아이콘의 모니터 실루엣 안에 '**DS**' 로고를 추가하여 시각적 정체성을 구축했습니다.

## [v1.4.0] - 2026-04-03
### 🚀 Improved
- **CPU 측정 정밀도 고도화**: 0.1초 단위의 스냅샷 측정 방식에서 호출 간격(1초) 사이의 **전체 평균 점유율**을 계산하는 방식으로 개선하여 작업 관리자와의 데이터 일치도를 높임.
- **범용 윈도우 GPU 모니터링(iGPU 포함)**: 윈도우 성능 카운터(Performance Counter) 연동을 통해 인텔 내장 그래픽, NVIDIA, AMD 모든 종류의 GPU 점유율을 수집할 수 있도록 수집기 개편.
- **네트워크 단위 체계 변경 (Bit 기준)**: 기존 Byte 단위(MB/s, KB/s)에서 네트워크 표준인 Bit 단위(**Mbps, kbps**)로 표시 단위를 변경하여 직관적인 속도 모니터링 지원.

### 🛠 Fixed
- **의존성 라이브러리 설치 오류 해결**: `Pillow` 등 누락된 패키지를 `requirements.txt` 기반으로 설치 완료하고 정상 구동 확인.

### 📋 Documentation
- **환경 구축 가이드 추가**: `README.md`에 파이썬 클라이언트 설치(`pip install -r requirements.txt`) 및 실행 방법을 상세히 기술하여 사용자 편의성 증대.

## [v1.3.0] - 2026-04-02
### 🚀 Added
- **하드웨어 버튼 및 부저 통합**: 4개의 버튼과 부저(Pin 7)를 지원하는 입력/피드백 시스템 구축.
- **화면 180도 회전(Toggle Flip)**: BTN1 조작으로 하드웨어와 소프트웨어가 동시에 회전하며, 물리적 배치에 맞춰 데이터 출력 순서를 **4-3-2-1**로 자동 반전시키는 지능형 매핑 적용.
- **OLED 밝기 조절 지원**: BTN2 조작으로 25%/50%/75%/100% 밝기를 순환하며, 도움말 창에 **4단계 사각형(◽️◼️) 비주얼 가이드** 실시간 표출.
- **설정 영구 저장(NVS Persistence)**: 화면 회전 각도 및 밝기 값을 내장 플래시에 기록하여 재부팅 시 자동 로드.
- **인터랙티브 도움말 모드**: BTN4 조작으로 각 버튼 기능 및 현재 설정 상태(각도/밝기)를 OLED 화면에 상시 요약 표시.

### 🛠 Fixed
- **U8G2 단일 공유 버퍼 버그 수정**: 모든 화면이 동일한 데이터로 덮어씌워지던 렌더링 결함을 구조적으로 해결 (그리기 직후 즉시 방출 방식 복구).
- **화면 깜빡임(Flicker) 제거**: 데이터 미수신 상태에서 매 루프 전송을 방지하고 상태 변화 감지 시에만 1회성 안내 화면 전송 로직 적용.
- **BLE 비동기 안정성 강화**: 스레드 세이프 공유 변수(`volatile`) 적용으로 데이터 수신 정지 현상 방지.

## [v1.2.1] - 2026-03-28
### 🚀 Improved
- **부팅 시 대기 화면 안내 메시지 추가**: 파이썬 클라이언트가 실행되지 않았을 때 OLED가 빈 상태로 남지 않도록 "PLEASE RUN PC CLIENT APP" 안내 메시지가 4개 OLED에 걸쳐 즉시 표시되도록 개선.
- **상태 관리 로직 최적화**: 펌웨어 내 `lastState` 초기값을 `-1`로 설정하여 부팅 직후 첫 화면 렌더링을 보장하고, BLE 연결 상태 변화에 따른 유연한 화면 전환 로직 구현.

## [v1.2.0] - 2026-03-27
### 🚀 Enhanced
- **실시간 네트워크 통합 모니터링**: 기존 전력 수치를 대체하여 다운로드/업로드 속도를 분리 측정 및 표시 루틴 구현.
- **OLED 고속 렌더링 최적화 (Dirty Checking)**: 섀도 버퍼를 통해 변경된 페이지만 전송하는 스마트 플러시 도입 (I2C 전송 대역폭 절감).
- **하드웨어 드라이버 고도화**: 
  - I2C 클럭 **800kHz** 상향 조정.
  - ESP32-C3 전용 **Fast GPIO Direct-writing** 기법 적용으로 SW I2C 성능 극대화.
- **UI/UX 디자인 리뉴얼**: 
  - 7-Segment 스타일 폰트 및 `profont15` 적용으로 가독성 및 세련미 향상.
  - 모든 대시보드 화면을 '라벨+수치+게이지 바' 레이아웃으로 통일.
- **트레이 아이콘 시인성 개선**: macOS 다크 모드에 최적화된 **투명 배경 + 흰색 컴퓨터** 실루엣 디자인 적용.
- **전용 주석 및 문서화**: 모든 소스 코드에 핵심 로직 및 최적화 기법에 대한 상세 한글 주석 추가.
- **MetricKit 기술 검토 완료**: 실시간 모니터링 목적에 맞지 않음을 검증하고 현재의 정밀 수집 방식 유지 결정.

## [v1.1.0] - 2026-03-26
### 🚀 Added
- **macOS(M2/M3) 정밀 지표 매칭**: `psutil` 및 시스템 파티션 분석을 통해 상용 앱(`mac stats`)과 동일한 수준의 RAM/Disk 수치 동기화 성공.
- **Apple Silicon GPU 실시간 모니터링**: `ioreg` 성능 지표 파싱을 통한 GPU 가동률 추출 로직 구현.
- **실시간 센서 스트림 파서**: 백그라운드에서 `powermetrics` 데이터를 실시간으로 낚아채는(Hooking) 스레드 파서 클래스(`PowerMetricsReader`) 도입 (1Hz 갱신).
- **BLE 통신 인프라 안정화**: UART 서비스 기반의 비동기 전송 프로토콜 구축 및 자동 재접속(Auto-Reconnect) 로직 강화.
- **UI/UX 고도화**: 대형 폰트(`maniac_tr`) 및 슬림 프로그레스 바를 적용한 GPS_DASH 스타일의 세련된 대시보드 레이아웃 적용.
- **디버깅 가시화**: 펌웨어 및 파이썬 클라이언트 전반에 실시간 데이터 수신 상태 로그(Serial/Terminal) 추가.
- **계획 수립**: `PLAN.md`를 통한 차기 개발 옵션 구체화.
