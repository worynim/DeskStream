# 🚀 DeskStream Release Notes

## [v1.5.6] - 2026-04-01
### Improved (OLED_Wide_test: 버튼 입력 반응성 비약적 향상)
- **실시간 버튼 핸들링 강화**: 화면 그리기(Rendering) 도중이나 프레임 대기(Frame Pacing) 구간에서도 버튼 입력을 지속적으로 스캔하여, 어느 시점에서도 즉각적인 화면 플립과 비프음 피드백이 가능하도록 로직을 재설계했습니다.

## [v1.5.5] - 2026-04-01
### Added (OLED_Wide_test: 180도 화면 플립 및 햅틱 피드백 도입)
- **180도 화면 회전 (Flip) 기능**: 1번 버튼(GPIO 1)을 통해 전체 디스플레이를 180도 뒤집는 기능을 구현했습니다.
- **다이나믹 디스플레이 스왑**: 화면이 뒤집힐 때 가상 캔버스의 구역 슬라이싱 로직을 역순(`1-2-3-4 -> 4-3-2-1`)으로 자동 전환하여, 가로 방향 애니메이션(Ticker 등)의 연속성을 완벽하게 유지합니다.
- **부저 피드백 및 NVS 저장**: 버튼 조작 시 3000Hz 비프음을 추가하고, 플립 상태를 비휘발성 메모리(Preferences)에 저장하여 재부팅 시 설정을 복원합니다.
- **성능 최적화**: I2C 클럭을 800kHz로 상향 조정하고, 지연(Pause) 시간 중에도 버튼 반응성을 유지하도록 `checkButtons()` 핸들러를 `smartDelay`에 통합했습니다.

## [v1.5.4] - 2026-04-01
### Added (Sound_Visualizer_4_OLEDs: 햅틱 부저 및 180도 스마트 회전 시스템 도입)
- **오디오 피드백 적용**: 캘리브레이션 트리거 시 짧은/긴 진입음에 대한 직관적인 펄스 제어(`beep()`) 적용.
- **180도 스마트 플립 기능**: 1번 핀 버튼 클릭 시 OLED 데이터 하드웨어 출력을 180도 회전시킴과 동시에, 가상 캔버스의 구역 슬라이싱 로직을 역순(`1-2-3-4 -> 4-3-2-1`)으로 자동 스왑하여 파형의 연속적 흐름 유지.
- **NVS 저장소 구축**: 변경된 플립 상태를 비휘발성 메모리(Preferences)에 실시간 캐싱하여 재부팅 시에도 표시 상태 완벽 보존.

## [v1.5.3] - 2026-04-01
### Added (GPS_DASH: 버튼 피드백 부저 추가)
- **버튼 조작음(Buzzer) 피드백 추가**: 7번 핀에 패시브 부저 펄스 제어(`beep()`) 로직을 추가하여, 짧은 클릭(Short)과 길게 누르기(Long) 조작 시 직관적인 청각적 피드백 제공.
- **프로젝트 간 일관성 유지**: `Smart_Info_Station`의 `beep()` 함수 구조를 `GPS_DASH`에 동일하게 복제하여 피드백 스니펫의 범용성 확보.

## [v1.5.2] - 2026-04-01
### Added & Fixed (Smart_Info_Station: 버튼 피드백 및 UI 매핑 고도화)
- **버튼 조작음(Buzzer) 피드백 추가**: 7번 핀에 패시브 부저 펄스 제어(`tone()`) 로직을 추가하여, 짧은 클릭(Short)과 길게 누르기(Long) 조작 시 청각적 피드백 제공.
- **직관적인 도움말 매핑 고정**: Flip Mode 설정으로 인해 화면이 교차(Swap) 출력되더라도, '버튼 설정 가이드 페이지'는 사용자의 물리적인 버튼 위치와 일치하도록 고정 매핑되게 예외 처리 적용.
- **Flip 전환 잔상 완전 제거**: 화면 회전 버튼 누름 즉시 4개의 OLED 하드웨어 버퍼를 일제히 초기화(`clearBuffer`)하고 전체 화면을 강제 재렌더링하여 발생하던 디스플레이 잔상(Ghosting) 문제 해결.

## [v1.5.1] - 2026-03-28
### Improved (Stats_Monitor: 부팅 시 OLED 대기 화면 안내 시스템 도입)
- **클라이언트 실행 유도 UX 개선**: 보드 전원 투입 직후(또는 연결 대기 중) OLED가 비어 있지 않도록 "PLEASE RUN PC CLIENT APP" 안내 메시지가 4개 OLED에 걸쳐 즉시 표시되도록 개선.
- **초기 렌더링 응답성 강화**: 펌웨어 내 상태 머신(`lastState = -1`)을 적용하여 부팅 및 재접속 시의 시각적 피드백 속도 최적화.

## [v1.5.0] - 2026-03-28
### Improved (Smart_Info_Station: 성능 최적화 및 스마트 루핑 도입)
- **화면 깜빡임 제로(Flicker-Free)**: 이중 버퍼 정제 로직 최적화를 통해 페이지 전환 시의 깜빡임을 제거하고 부드러운 UI 전환 구현.
- **스마트 자동 루핑 시스템**: 데이터 수집 중일 때 페이지 전환을 자동 유예하여, 수시로 변하는 정보의 수집 완료를 보장하는 지능형 루핑 도입.
- **위젯별 독립 수집 주기 관리**: 글로벌 타이머의 한계를 극복하고 각 서비스(금융, 날씨 등)별 독립적인 갱신 주기를 철저히 준수하도록 엔진 개선.
- **사용자 편의성 강화**: 폰트 중앙 관리 매크로(`MAIN_NUM_FONT`) 도입 및 기본 밝기(25%) 최적화를 통한 번인 방지.

## [v1.4.0] - 2026-03-28
### Added (Smart_Info_Station: 4버튼 내비게이션 및 사용자 인터페이스 확장)
- **4버튼 독립 제어 시스템 구축**: `GPS_DASH`의 버튼 아키텍처를 이식하여 4개 버튼에 화면 회전, 밝기 조절, 페이지 전환, 데이터 새로고침 기능을 독립적으로 할당.
  - **BTN 1**: 짧게(Short) - **화면 회전(Flip)** / 길게(Long) - (Reserved)
  - **BTN 2**: 짧게(Short) - **밝기(Contrast)** 4단계 조절 / 길게(Long) - (Reserved)
  - **BTN 3**: 짧게(Short) - (Reserved) / 길게(Long) - **자동 루핑(Auto Loop)** ON/OFF
  - **BTN 4**: 짧게(Short) - **페이지 전환(Next)** / 길게(Long) - **데이터 강제 새로고침**
- **인터랙티브 도움말 시스템**: 버튼별 현재 기능을 OLED에서 즉시 확인할 수 있는 도움말 페이지(**Page 5**) 및 IP 정보(**Page 4**) 배치.
- **사용자 설정 영속성**: 회전, 밝기, 루핑 설정을 `Preferences`에 저장하여 재부팅 후에도 개인화된 설정 유지.
- **웹 대시보드 확장**: 자동 루핑 설정을 웹 설정 UI 카드 내부로 통합하여 실시간 원격 제어 지원.

## [v1.2.0] - 2026-03-27
### Enhanced (Stats_Monitor 프로젝트 고도화)
- **실시간 네트워크 통합 모니터링**: 의미가 고정적인 전력 수치 대신, 생동감 넘치는 다운로드/업로드 속도 실시간 측정 로직 도입.
- **OLED 렌더링 물리 엔진 최적화**: 
  - **Shadow Buffer** 기반의 Dirty Checking 기법을 수동으로 구현하여 I2C 전송 대역폭 효율 극대화.
  - ESP32-C3 하드웨어 레지스터 직접 제어(Register-direct)를 통한 소프트웨어 I2C 성능 획기적 개선.
  - I2C 버스 클럭 **800kHz** 상향 조정.
- **가독성 기반 UI/UX 리뉴얼**: 
  - 7-Segment 스타일 및 `profont15` 적용으로 세련된 대시보드 시각화 구현.
  - 모든 상태 지표를 '라벨+수치+게이지 바' 레이아웃으로 통일하여 통합된 디자인 경험 제공.
- **시스템 모니터 트레이 아이콘 최적화**: macOS 투명 상태 바에 최적화된 화이트 실루엣 아이콘 적용.
- **코드 무결성 강화**: 모든 프로젝트 소스 코드 및 최적화 기법에 대한 상세 기술 주석 추가 완료.

## [v1.1.0] - 2026-03-26
### Added (Stats_Monitor 프로젝트 강화)
- **macOS(M2/M3) 정밀 지표 매칭**: `psutil` 및 시스템 파티션 분석을 통해 상용 앱(`mac stats`)과 동일한 수준의 RAM/Disk 수치 동기화 성공.
- **Apple Silicon GPU 실시간 모니터링**: `ioreg` 성능 지표 파싱을 통한 GPU 가동률 추출 로직 구현.
- **실시간 센서 스트림 파서**: 백그라운드에서 `powermetrics` 데이터를 실시간으로 낚아채는(Hooking) 스레드 파서 클래스(`PowerMetricsReader`) 도입 (1Hz 갱신).
- **BLE 통신 인프라 안정화**: UART 서비스 기반의 비동기 전송 프로토콜 구축 및 자동 재접속(Auto-Reconnect) 로직 강화.
- **UI/UX 고도화**: 대형 폰트(`maniac_tr`) 및 슬림 프로그레스 바를 적용한 GPS_DASH 스타일의 세련된 대시보드 레이아웃 적용.
- **디버깅 가시화**: 펌웨어 및 파이썬 클라이언트 전반에 실시간 데이터 수신 상태 로그(Serial/Terminal) 추가.
- **계획 수립**: `PLAN.md`를 통한 차기 개발 옵션 구체화.

  - `Stats_Monitor`: 시스템 상태 모니터 (CPU, GPU, RAM, SSD 등)

- 루트 디렉토리 `README.md` 생성 및 프로젝트 가이드라인 구축.
