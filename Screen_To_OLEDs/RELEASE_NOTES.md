# Release Notes - Screen_To_OLEDs

## [V1.0.0] - 2026-04-09

### 🚀 초기 릴리즈 (Final MVP 완료)

PC(macOS) 화면을 4개의 OLED 매트릭스로 구성된 DeskStream 하드웨어에 실시간 스트리밍하는 시스템이 완성되었습니다.

#### 🆕 추가된 기능
- **macOS 스트리머 기능 (`screen_streamer.py`)**:
    - **UI 개선**: 레이아웃(가로/세로) 설정을 라디오 버튼으로 변경하여 직관성 향상.
    - **캡처 위치 기억**: 스트리밍 중단 후 재시작 시에도 마지막 드래그 위치를 자동으로 복구.
    - **디더링 최적화**: Floyd-Steinberg Dithering 모드 선택 시 Contrast 기본값을 2.0으로 자동 상향하여 가독성 강화.
    - **오버레이 유지**: 스트리밍 정지 상태에서도 캡처 영역 오버레이를 유지하여 미리 위치 조정 가능.
    - **투명 윈도우 대응**: macOS 특유의 캡처 버그를 회피하는 4-선 독립 윈도우 오버레이 방식 적용.
    - **Portrait 지원**: 64x512 세로 모드 캡처 및 자동 회전 전송 로직 구현.
    - **실시간 제어**: B&W 모드, Contrast, FPS 등을 스트리밍 중 중단 없이 실시간 변경 가능.
- **ESP32 펌웨어 기능**:
    - **인터랙티브 도움말**: BTN 2 롱프레스 시 나타나는 도움말 화면에 밝기 상태를 사각형 심볼(■■□□)로 시각화.
    - **지능형 화면 배치**: 도움말 화면 출력 시 하드웨어 버튼 위치와 일치시키기 위해 전체 플립(Flip) 설정을 일시 무시하고 정방향으로 고정 출력.
    - **스마트 IP 표시**: 대기 화면 시 IP 주소의 각 숫자를 OLED 4개에 나누어 크게 표시 (물리 경계선 가독성 해결).
    - **고정밀 부저**: Ticker 기반 비차단 로직을 적용하여 loop 지연과 상관없이 20~30ms의 경쾌한 피드백 제공.
    - **원버튼 밝기 제어**: BTN2 하나로 밝기 단계(1 > 50 > 100 > 240)를 순환하며 조절 가능.
    - **플립 동기화**: BTN1 롱프레스 시 즉시 현재 UI 화면의 순서까지 반전되도록 개선.
    - **자동 복구**: UDP 스트림 3초 이상 누락 시 자동으로 IP 안내 화면으로 복귀.

#### 🔧 개선 및 버그 수정
- **UI 반응성**: 밝기 조절 시 도움말 화면의 비주얼 바가 실시간으로 갱신되지 않던 문제 해결.
- **비트 매핑**: SSD1306 페이지 모드와 LovyanGFX 메모리 구조 불일치 해결 (Row-major, MSB-first 최적화).
- **네트워크 안정성**: 프레임 ID 기반 청크 재조립 및 타임아웃 관리 로직 강화.

#### 📋 사용 안내
- **실행**: `python screen_streamer.py`
- **조작**: 
    - 캡처 영역 중앙의 **✥** 핸들을 드래그하여 위치 이동.
    - 하드웨어 BTN2를 짧게 눌러 밝기 조절, 길게 눌러 도움말 확인.
